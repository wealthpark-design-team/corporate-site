# WealthPark コーポレートサイト リニューアルプロジェクト

## 📋 プロジェクト概要

WealthPark株式会社のコーポレートサイトにおける、デザイン・UI・技術スタックの全面刷新プロジェクト。

## 🎯 リニューアルの目的

### 主要目標
- **デザイン・UIの刷新**: モダンで洗練されたビジュアルデザインへの移行
- **技術スタックの近代化**: 2025年のベストプラクティスに基づいた技術選定
- **ブランディングの強化**: #1a1a1aをブランドカラーとした統一感のあるデザインシステム

### 重要な制約
- **コンテンツは完全に同一**: 既存サイトの文言を一字一句変更せず、そのまま使用
- **段階的な移行**: まずスタティックサイトとして構築し、後にWordPress APIを連携

## 🏗️ 技術スタック

### フロントエンド
- **Next.js 16** (App Router) - React フレームワーク
- **TypeScript** - 型安全性の確保
- **Tailwind CSS v4** - ユーティリティファーストCSS
- **React 19** - 最新のReactバージョン

### デザインシステム
- **ブランドカラー**: `#1a1a1a`
- **フォント**: システムフォント + 日本語最適化
  - `-apple-system, BlinkMacSystemFont, "Segoe UI", "Noto Sans JP", "Hiragino Kaku Gothic ProN", "Hiragino Sans", Meiryo, sans-serif`
- **スペーシング**: 8pxベースのシステマチックな余白設計
- **タイポグラフィ**: Fluid typography (clamp使用)

### デプロイ（予定）
- **Vercel** - Next.js最適化済みホスティング

### CMS連携（今後）
- **WordPress REST API** - ニュース・ブログ記事の取得
- **Kinsta** - 既存のWordPressホスティング

## 🎨 デザインコンセプト

### デザイン参考
- **LayerX.co.jp**: ミニマル・クリーン・モダンなデザイン
- **白ベース**: 余白を活かしたスペーシング
- **#1a1a1a**: ブランドブラックをアクセントに使用

### デザイン原則
1. **ミニマリズム**: 必要最小限の要素で情報を伝える
2. **余白の活用**: 十分なホワイトスペースで可読性を向上
3. **タイポグラフィ重視**: 大胆な見出しと読みやすい本文
4. **レスポンシブ**: モバイルファーストの設計
